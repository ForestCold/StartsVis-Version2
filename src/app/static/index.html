STARTS Based

STARTS is a static class-level regression test selection tool for Maven-based Java programs. STARTSVIS is a d3-based visualize and analysis system for STARTS. See more about STARTS.

Types Filter

STARTSVIS help you classify types into classes, tests and libraries. Automatically filtering libraries that have nothing to do with Regression Test Selection. Also support user-guided types selecting process.

Multi-module Supported

STARTSVIS supports analyzing projects with multi modules. Automatically parsing the pom.xml file in the project's directory. Hash Map and Canvas are used to accelerate types analysis and visualization process.

Dependencies Visualization

STARTSVIS provides graph and tree components for types dependency visualization. Help users better understand about the Regression Test Selection strategy in STARTS.
